🛠️ 实施步骤

  1. 安装 Ollama — 一键安装并拉取模型
    curl -fsSL https://ollama.com/install.sh | sh
    ollama pull llama3.3:8b（或 mistral-small-3、qwen3:4b）
  2. 安装 OpenClaw — 配置 Ollama 作为后端
    openclaw config set models.providers.ollama.baseUrl "http://127.0.0.1:11434"
    Mac Mini 常驻运行，无需每次启动
  3. 配置 SOUL.md + AGENTS.md — 定义 AI 角色、行为规范、记忆文件位置
  4. 安装 n8n（可选） — 结构化工作流自动化
    npm install n8n -g && n8n start
    n8n v2.0 已支持 AI Agent 节点，本地跑在 Mac Mini 上
  5. 设置定时任务 — cron jobs 维持 AI agent 持续运行
    晨间简报（morning briefing）、定期研究任务、安全监控等
  6. 连接飞书/日历/邮件 — OpenClaw 技能实现全平台集成
