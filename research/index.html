📚 历史发现摘要

高频主题发现

  • Mac Mini M4 统一内存架构：GPU 可访问全部内存池，无传统 VRAM 上限
  • M4 16GB：7B-8B Q4 模型 28-35 tok/s；Mistral Small 3 7B 达到 50 tok/s
  • OpenClaw + n8n 互补架构：OpenClaw 做决策，n8n 做执行
  • Home Assistant 2026.3+ 原生 Ollama 集成，Settings → Devices & Services → Add Ollama
  • ClawBox vs Mac Mini TCO：Mac Mini 快 3 倍，便宜 ~€530
  • 本地 Whisper 语音识别是"无脑选"：质量好，延迟可接受
  • Home Assistant + 本地 LLM 存在 token 爆炸问题（8k tokens/命令）→ 解决：自定义 Modelfile
  • Obsidian Git 插件：自动 commit + push 到 GitHub，双重备份
  • rsync Time Machine 风格：硬链接增量快照，普通文件夹格式

⭐ 可建站方案汇总

  • Home Assistant + Obsidian 自动日记（午夜触发，环境数据自动填充）

  • OpenClaw 全业务栈监控（多目录 + cron + AI 分析）
  • n8n + Ollama 本地 RAG（文档摘要 + 知识问答）
  • AI 情报站（Miniflux → n8n → Ollama → 飞书）
