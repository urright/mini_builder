💡 方案简介

Mac Mini M4 可以作为强大的家庭文件服务器，用 SMB/NFS 协议为全家人提供集中式文件存储、Time Machine 备份目标，以及远程访问能力。以太网直连速度可达 10Gbps，比任何网盘都快，且完全私有、零订阅。

📋 实施步骤

1
准备外接存储：Mac Mini M4 基础版 256GB/512GB SSD 不足以存媒体文件，建议外接 NAS 级 HDD。推荐配置：2TB NVMe SSD（系统高速存储）+ 10-20TB HDD（冷数据归档）。使用 USB4/Thunderbolt 外壳（如 ACASIS 40Gbps M.2 NVMe，约 $80）。
ACASIS 40Gbps M.2 NVMe SSD 外壳 - Amazon
2
启用 SMB 文件共享：macOS 系统设置 → 通用 → 共享 → 开启「文件共享」。SMB 协议支持全平台（Mac/Windows/Linux/iOS/Android）。建议创建专用用户账户，限制访问权限。
sudo launchctl load -w /System/Library/LaunchDaemons/com.apple.smbd.plist
3
配置 Time Machine 目标（可选）：Mac Mini 可以作为整个家庭所有 Mac 的 Time Machine 备份服务器。系统设置 → Time Machine → 选择磁盘 → 找到 Mac Mini。推荐用 HDD 而非 SSD 节省成本。
sudo tmutil setdestination -a smb://username@macmini.local/BackupDrive
4
远程访问：Tailscale 或 Cloudflare Tunnel：
Tailscale（推荐，简单）：安装后自动建立加密 VPN，macOS 客户端可直接挂载 SMB。
brew install tailscale → sudo tailscaled install-system-daemon
Cloudflare Tunnel：免费，无需开放端口，通过 cf-tunnel 暴露文件服务。
Tailscale macOS 下载
5
JetKVM 远程管理（可选，物理访问场景）：JetKVM 是硬件 KVM-over-IP 设备，可在远程完全控制 Mac Mini 的显示器和键盘，适合没有现有桌面环境时初始配置。
JetKVM - Amazon（$60，约 $45）
6
设置自动备份脚本：用 rsync 或 rclone 定期将重要数据同步到外接 HDD，并可选择性地同步到 Google Drive/Backblaze B2 作为异地备份。
rsync -avz --delete /Users/jeremy/Documents/ /Volumes/BackupDrive/Documents/

📊 效果量化数据

  • 存储成本：20TB NAS 级 HDD（约 $250）vs 20TB Synology NAS（$500+），Mac Mini 方案节省 50%+
  • 速度：以太网 10Gbps ≈ 1.25GB/s，比任何云存储快 10-50 倍
  • 功耗：Mac Mini M4 空闲功耗 ~6W，满载 ~30W；Synology DS923+ 约 30W/45W（更多）
  • 月电费：30W × 24h × 30天 ≈ 21.6kWh → 约 $3-5/月（加州电价）
  • 订阅节省：告别 iCloud 2TB（$12.99/月）、Dropbox、Google One，每年节省 $156-468
