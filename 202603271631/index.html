📖 使用指南

核心功能

  • 日历管理: 创建/查询/修改飞书日程，自动计算最佳会议时间
  • 任务管理: 创建任务、设置截止日期、分配负责人、追踪进度
  • 消息处理: 读取飞书消息、搜索历史记录、提取关键信息
  • 文档处理: 创建/编辑飞书文档、总结内容、提取数据
  • 定时任务: 设置 cron 任务定期执行，如每日报表、每周总结

进阶用法

  • 多技能组合: 组合多个技能完成复杂任务，如"查日程 → 发消息 → 更新任务"
  • 心跳主动通知: 配置 HEARTBEAT.md 让 OpenClaw 主动推送重要信息
  • 子 Agent 并行: 大任务拆分成多个子 agent 并行处理
  • 记忆系统: 利用 MEMORY.md 和 daily notes 建立长期记忆

最佳实践

  • 用清晰、具体的指令，AI 理解更准确
  • 复杂任务拆分成多个简单步骤
  • 定期更新 memory 文件让 AI 更好了解你的偏好
  • 设置 HEARTBEAT.md 自动化 routine 任务
