🔧 实施方案

第一步：安装 UTM（虚拟机）

# 方法1: App Store 下载 (免费)
# 搜索 "UTM" 安装

# 方法2: Homebrew
brew install --cask utm

第二步：下载 Home Assistant OS

# 下载 Home Assistant OS QCOW 镜像
# 访问: https://www.home-assistant.io/installation/macintosh/
# 下载 haos_ova-*.qcow2.tar.xz

# 解压
tar -xvf haos_ova-*.qcow2.tar.xz

第三步：创建 UTM 虚拟机

# 打开 UTM
# 1. 点击 "Create New Virtual Machine"
# 2. 选择 "Emulate" -> "Linux"
# 3. 配置:
#    - RAM: 4096 MB (4GB)
#    - CPU: 4 cores
#    - Storage: 64 GB
# 4. 加载下载的 qcow2 镜像
# 5. 启动虚拟机

第四步：配置 Home Assistant

# 虚拟机启动后
# 1. 等待 Home Assistant 启动完成 (约 10-20 分钟)
# 2. 在浏览器访问: http://homeassistant.local:8123
# 3. 按提示创建管理员账户
# 4. 开始添加设备

第五步：配置自动化示例

# 示例: 日出时自动开灯
automation:
  - alias: "日出开灯"
    trigger:
      platform: sun
      event: sunrise
    action:
      service: light.turn_on
      entity_id: light.living_room

# 示例: 离家模式
  - alias: "离家模式"
    trigger:
      platform: device
      device_id: xxx
      type: leaving
    action:
      - service: light.turn_off
        target:
          entity_id: all
      - service: climate.set_temperature
        data:
          temperature: 18

✨ 方案优势

  • 统一控制: 一个界面控制所有品牌设备，Alexa、Google Home、小米、米家等
  • 本地运行: 不依赖云服务，断网也能用，保护隐私
  • 无限自动化: 用 YAML 或可视化编辑器创建复杂自动化规则
  • 开源社区: 庞大的插件生态，HACS 社区集成
  • Mac Mini M4 性能: 虚拟机运行流畅，CPU 占用低
