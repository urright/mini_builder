🔧 实施方案

第一步：安装 Docker

# Homebrew 安装 Docker
brew install --cask docker

# 或者去 https://docker.com 下载 Desktop 版

第二步：安装 n8n

# 方法1: Docker Compose (推荐)
# 创建 docker-compose.yml
version: '3'
services:
  n8n:
    image: n8nio/n8n
    ports:
      - "5678:5678"
    volumes:
      - ~/.n8n:/home/node/.n8n
    environment:
      - N8N_DEFAULT_BINARY_DATA_MODE=filesystem

# 启动
docker-compose up -d

第三步：访问 n8n

# 浏览器打开
http://localhost:5678

# 首次使用创建账户

第四步：创建第一个工作流

# 示例: 收到邮件 → 创建任务 → 发送 Slack 通知

# 1. 点击 "New Workflow"
# 2. 添加 Trigger: Email (IMAP)
# 3. 添加 Node: Notion (创建任务)
# 4. 添加 Node: Slack (发送消息)
# 5. 连接并激活

📖 使用指南

  • 工作流模板: n8n 社区有大量预制模板，直接导入使用
  • 条件分支: 用 IF 节点做条件判断，复杂逻辑也能可视化
  • 代码节点: 需要自定义时，可用 JavaScript/Python 写代码
  • 错误处理: 设置 Error Trigger，自动通知工作流失败
  • 定时触发: 用 Schedule Trigger 做定时任务，如每日报告
