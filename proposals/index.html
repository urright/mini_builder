Mac Mini 全自动备份系统

安全 自托管

rsync Time Machine 风格增量快照脚本 + launchd 定时调度 + 外挂硬盘。遵循 3-2-1 原则（3份副本、2种介质、1份异地）。

MB级
增量大小
∞
版本快照
3-2-1
备份原则

核心脚本

#!/bin/bash
TIMESTAMP=$(date +"%Y-%m-%dT%H-%M-%S")
SOURCE="/Users/jeremyren"
DEST="/Volumes/BackupMini/snapshots"
EXCLUDES="/Volumes/BackupMini/rsync-excludes.txt"

# 创建快照，使用硬链接节省空间
rsync -avPh --delete \
  --exclude-from=$EXCLUDES \
  --link-dest=$DEST/latest \
  $SOURCE/ $DEST/$TIMESTAMP/

# 更新 latest 软链接
rm -f $DEST/latest
ln -s $DEST/$TIMESTAMP $DEST/latest

预期效果

  • 每次备份只上传变更文件（MB 级 vs GB 级）
  • 任意时间点可恢复，硬盘满了删旧快照
  • 配合 Time Machine = 双重保护
