家庭服务器 UniFi AdGuard Homebridge 网络管理 可建站

服务详解

AdGuard Home — 全局广告拦截

  • DNS 层面拦截：所有设备通过 AdGuard DNS 过滤，无需在每个设备安装
  • 隐私保护：阻止追踪器，保护家庭网络隐私
  • 自定义规则：支持订阅 AdGuard DNS filter、EasyList 等规则
  • 查询日志：查看每个设备的网络请求，发现异常连接

Homebridge — 智能家居桥接

  • HomeKit 兼容：让非 HomeKit 设备（如小米、Yeelight）出现在苹果家庭 App
  • 插件生态：数千个插件支持各种智能设备协议
  • Siri 语音控制：通过 Homebridge 接入的设备支持 Siri 语音命令
  • 自动化场景：配合 Home Assistant 实现复杂自动化

UniFi Network — 企业级网络管理

  • UniFi Controller：管理 UniFi 路由器、交换机、AP
  • 流量监控：实时查看设备流量和网络状态
  • VPN 服务器：远程安全访问家庭网络
  • 端口转发 / 防火墙规则：精细化网络控制

实施步骤

第一步：安装 Homebrew

macOS 包管理工具，是安装所有服务的入口：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
第二步：安装 AdGuard Home

通过 Homebrew 安装并启动 AdGuard Home：

brew install adguardhome
brew services start adguardhome

安装后访问 http://localhost:3000 进行初始化配置，将路由器 DNS 指向 Mac Mini IP。

第三步：安装 Homebridge

Node.js 环境 + Homebridge 核心，通过 Homebrew：

brew install node
npm install -g --unsafe-perm homebridge homebridge-config-ui-x
hb-service install

访问 http://localhost:8581 进入 Web UI，配置插件和智能设备。

第四步：安装 UniFi Network Controller

仅在使用 UniFi 设备时需要，非必需：

brew install macminivault/taps/unifi-network-controller
brew services start unifi-network-controller

访问 https://localhost:8443 初始化 UniFi 网络。

第五步：配置 macOS 节能选项（24/7 运行关键）

关闭盖子时禁止 Mac Mini 睡眠，确保服务持续运行：

# 阻止盖子关闭时睡眠（需要保持连接显示器）
sudo pmset -a sleep 0
sudo pmset -a hibernatemode 0
sudo pmset -a disablesleep 1

# 或者在"系统设置 > 节能"中：
# - 关闭"使硬盘进入睡眠"
# - 设置"电源适配器：永不断开"
第六步：配置静态 IP 和路由器 DNS

给 Mac Mini 分配固定 IP（如 192.168.1.100），在路由器 DHCP 设置中将 DNS 服务器指向 Mac Mini IP。

效果量化

  • 功耗：空闲 6-8W，满载 20-30W（vs 传统服务器 100-300W）
  • 噪音：日常运行几乎无声（仅高负载时风扇轻微转动）
  • 广告拦截率：AdGuard Home 配合 EasyList + AdGuard DNS filter，拦截率 85-95%
  • 设备兼容性：Homebridge 插件库支持 2000+ 设备型号
  • 总成本：软件全部免费，仅需 Mac Mini 硬件成本

硬件配置推荐

  • 最低：Mac Mini M4 16GB RAM（AdGuard + Homebridge 够用）
  • 推荐：Mac Mini M4 24GB RAM（同时跑 UniFi + Plex 媒体服务）
  • 发烧友：Mac Mini M4 Pro 48GB RAM（虚拟化、容器、多种服务并行）
  • 存储：基础款 256GB 够用，服务占用约 20-30GB；大量媒体文件建议外接 NAS

与 Home Assistant 的关系

本方案专注网络层服务（DNS、VPN、网络设备管理），Home Assistant 专注设备控制层（灯光、传感器、自动化逻辑）。两者完全互补，可以同时运行在一台 Mac Mini 上：Homebridge 负责将设备接入 HomeKit，Home Assistant 负责设备联动和自动化。
