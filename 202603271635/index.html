🔧 实施方案

第一步：选择 RSS 服务

# 推荐 Newsblur (免费额度足够)
# https://newsblur.com
# 注册账号，免费版支持 64 个订阅源

# 其他选择
# - Inoreader: 免费 150 个源
# - Feedly: 免费 100 个源
# - The Old Reader: 免费无限（社交功能）

第二步：安装 Reeder

# App Store 搜索 "Reeder"
# 价格: ¥68 (一次性买断)
# 支持 Mac + iOS 同步

# 替代: NetNewsWire (免费开源)
brew install --cask netnewswire

第三步：配置 Reeder + Newsblur

# 1. 打开 Reeder
# 2. 添加账户选择 Newsblur
# 3. 输入 Newsblur 用户名密码
# 4. 开始同步

第四步：订阅优质来源

# 推荐的 RSS 来源类型
# 技术博客: 知名工程师博客、官方博客
# 行业新闻: 垂直领域头部媒体
# 工具更新: 你使用的工具官方发布
# 学术论文: arXiv、Google Scholar (用 Zotero connector)

# 避免
# - 社交媒体转帖 (重复内容多)
# - 标题党媒体 (质量差)
# - 更新过于频繁的源 (>10篇/天)

📖 使用指南

  • 每日限额: 设定每天最多阅读时间，如 45 分钟
  • 快速浏览: 用快捷键 J/K 跳转，Space 展开，不感兴趣直接 N
  • 稍后读: 用 Reeder 内置的 Readability 功能保存完整文章
  • 标签管理: 为不同来源打标签，如 #技术 #产品 #行业
  • 定期清理: 每月检查订阅源，移除低质量或长期不看的

✨ 方案优势

  • 算法退散: 不再被 TikTok/微博算法控制阅读内容
  • 信息可控: 自己决定看什么，不被流量标题党
  • 时间可控: 设定阅读时段和时长，不再沉迷
  • 长期积累: 有价值的文章保存稍后读，慢慢消化
  • 专注力提升: 减少信息碎片输入，提升深度阅读能力

⚠️ RSS 的坑与避免方法

  • 订阅源过多: 从 10-15 个开始，逐渐筛选。不活跃或重复内容的源每月清理一次。
  • 新闻 RSS 时效性差: 选择带完整内容的 RSS，而非只有摘要。推荐使用 Full Text RSS 提取全文。
  • 刷 RSS 上瘾: 设置每天固定阅读时间（如 20 分钟），用定时器控制。RSS 替代了 Twitter，不代表要 24/7 刷新。
  • 平台关停风险: Google Reader 已关闭过。推荐自建 FreshRSS 或使用 Inoreader 多平台同步。
  • Newsletter 转化: 很多 Newsletter 同时提供 RSS 订阅（如 Substack 付费文章也有免费 RSS），善用此技巧。
