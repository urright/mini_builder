发现 #093-A

主题: GitHub Actions 自托管 Runner 2026年3月起开始收费

来源:

  • Reddit — GitHub: Self-Hosted Action Runners will be billed from March 1, 2026
  • GitHub Community — Self-hosted runner minimum version enforcement v2.329.0
  • GitHub Blog — macOS 26 is now generally available for GitHub-hosted runners
  • Scaleway — Configure GitHub Actions Runner on Mac mini

核心数据:

  • 重大变化: 自2026年3月1日起，GitHub Actions 自托管 runner 开始收费（此消息在 Reddit 获得高热度，score 0.78）
  • 2026年1月1日起，GitHub 托管 runner 价格下调高达 39%（取决于机器类型）
  • 自托管 runner 最低版本要求：从 2026 年 3 月 16 日起，GitHub Actions 将阻止低于 v2.329.0 的自托管 runner 配置
  • macOS 26 runner 现已全面可用，支持 Apple Silicon (arm64) 和 Intel (x64)
  • macOS GitHub runner $0.08/分钟，自托管 Mac Mini 仅需电费（$0.05-0.1/天）
  • GitHub 托管 macOS 构建按分钟计费，自托管 Mac Mini 绕过此限制

实施要点:

  • 现有自托管 runner 用户需确保版本 ≥ v2.329.0（2025年10月15日发布）
  • macOS 26 runner 已GA：新环境包含最新 Xcode，可用于替代本地 runner 测试
  • 自托管 runner 适合有状态构建（缓存 Xcode derivedData），比 GitHub 托管 runner 更快
  • 推荐配置 LaunchDaemon 开机自启：`sudo ./svc.sh install`

效果量化:

  • 即使开始收费，Mac Mini 自托管 runner 成本仍远低于 GitHub 托管：电费 $0.05-0.1/天 vs $0.08/分钟（按 1000分钟/月 = $80/天 = 远超自托管成本）
  • macOS 构建分钟单价是 Linux 的 10 倍，自托管优势更显著

归属: 扩展方案 #202603281230（GitHub Actions 自托管 Runner）

  • Dev.to — The Homelab AI Stack in 2026: What Self-Hosters Are Actually Running
  • Tedium — Self-Hosting Tools: Still Worth Trying In 2026?

  • Tailscale 作为异地组网方案，取代 port forwarding
  • OpenClaw 以 service 方式 15W 常驻，与 n8n 协同工作

效果量化:
