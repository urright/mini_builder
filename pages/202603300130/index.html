🔧 实施步骤

安装 Homebrew

macOS 必备包管理器，简化软件安装流程。

/bin/bash -c "$(curl -fsSL https://brew.sh/install.sh)"

安装 Miniforge (Conda) — Python 3.12

用 Conda 隔离 Python 环境，避免与系统 Python 冲突。

bash Miniforge3-MacOSX-arm64.sh
source ~/.zshrc
conda activate base

创建独立环境并安装 Open WebUI

推荐 Python 3.12，Conda 环境隔离确保依赖干净。

conda create -n open-webui python=3.12
conda activate open-webui
pip install open-webui

拉取本地模型

推荐轻量模型开始，Mac Mini M4 16GB 流畅运行。

# 轻量入门（推荐 16GB 用户）
ollama pull llama3.2:3b

# 性能更优（推荐 24GB+ 用户）
ollama pull mistral-small-3:latest

# 高质量通用
ollama pull qwen3:4b

启动 Open WebUI

启动后访问本地地址，开始使用。

open-webui serve
# 访问: http://localhost:8080
