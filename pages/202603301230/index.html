📋 实施步骤

安装 Docker Desktop

Mac Mini M4 直接安装 Docker Desktop（Apple Silicon 版本），开启 Kubernetes 支持。Docker Desktop Settings → Resources → Memory 分配 4GB+，CPU 4核+。

brew install --cask docker

安装 Portainer（Web 可视化管理）

用 Portainer 替代命令行运维，Web UI 管理所有容器、镜像、网络、卷。

docker run -d \
  --name portainer \
  -p 9000:9000 \
  -p 8000:8000 \
  --restart always \
  -v /var/run/docker.sock:/var/run/docker.sock \
  -v portainer_data:/data \
  portainer/portainer-ce:latest

编写 docker-compose.yml（一键部署全家桶）

在一个目录管理所有服务，一个命令启动/停止整个 Homelab。

services:
  adguard:
    image: adguard/adguardhome
    container_name: adguard
    ports:
      - "53:53/tcp"
      - "53:53/udp"
      - "3000:3000"
    volumes:
      - adguard_data:/opt/adguardhome/work
    restart: unless-stopped

  n8n:
    image: n8nio/n8n
    container_name: n8n
    ports:
      - "5678:5678"
    volumes:
      - n8n_data:/home/node/.n8n
    restart: unless-stopped

  ollama:
    image: ollama/ollama
    container_name: ollama
    ports:
      - "11434:11434"
    volumes:
      - ollama_data:/root/.ollama
    restart: unless-stopped
    # Mac GPU 加速（Apple Silicon）
    deploy:
      resources:
        reservations:
          devices:
            - driver: metal
              count: all

  uptime-kuma:
    image: louislam/uptime-kuma
    container_name: uptime-kuma
    ports:
      - "3001:3001"
    volumes:
      - uptime_kuma_data:/app/data
    restart: unless-stopped

volumes:
  adguard_data:
  n8n_data:
  ollama_data:
  uptime_kuma_data:

部署 nginx-proxy-manager（反向代理 + 自动 SSL）

所有服务通过域名访问，自动获取 Let's Encrypt SSL 证书。

services:
  nginx-proxy-manager:
    image: jc21/nginx-proxy-manager
    ports:
      - "80:80"
      - "443:443"
      - "81:81"
    volumes:
      - npm_data:/data
      - npm_letsencrypt:/etc/letsencrypt
    restart: unless-stopped

配置 AdGuard Home 为网络 DNS

将路由器 DNS 指向 Mac Mini IP，所有设备自动享受广告拦截 + 隐私保护。AdGuard 实测拦截率 15-30%，每月拦截数万次广告请求。

开机自启 + 电源管理

macOS 系统偏好设置 → 用户与群组 → 登录项，添加 Docker Desktop。配合「能源」设置，关闭盖子不休眠（需接电）：

# 关闭盖子不休眠（接电时）
sudo pmset -a lidwake 1
sudo pmset -a sleep 0
# 或保持盖子打开，用 HDMI dummy plug 欺骗显示器

✅ Mac Mini Docker Homelab 优势

  • 功耗：5-15W（vs 传统 PC 服务器 100W+）
  • 噪音：几乎零噪音（被动散热 M4）
  • 电费：~$2-5/月（vs 云服务器 $20-50/月）
  • 数据完全私有，无订阅涨价风险
  • 一台机器：开发 + AI + 自动化 + 存储
  • macOS 原生支持 Docker Apple Silicon

⚠️ 注意事项

  • Mac Mini 无法像 TrueNAS 那样做 ZFS 存储池
  • 容器数据持久化需注意卷挂载路径
  • 无 ECC 内存，不适合 24/7 关键数据存储
  • 建议外接 NAS 或 HDD 做归档存储
💡 与「文件服务器」方案结合：Mac Mini 跑 Docker 服务，外接 NAS 做存储，是性价比最高的家庭基础设施方案。
