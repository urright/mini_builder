效果量化数据

  • API 成本节省: 完全本地运行，$0 API 费用，相比 OpenAI GPT-4o 节省 ~$100/月
  • 推理速度 (Mac Mini M4 16GB): Llama 3.3 8B Q4_K_M ~18-22 tok/s，Mistral Small 3 7B ~20-25 tok/s
  • 内存占用: Ollama + 向量数据库 + OpenClaw 后台运行，约 8-12GB
  • 隐私保证: 数据从未离开本地网络，满足 GDPR 等合规要求
  • 用户报告: 内容生产者节省 3-5h/周，研究者节省 10h+/周

⚙️ Ollama 配置（OpenClaw 使用本地模型）

在 OpenClaw 配置文件中指定 Ollama 作为 LLM 后端：

ollama.baseUrl = "http://127.0.0.1:11434"

实施步骤

  1. 安装 Ollama
    macOS: brew install ollama
    或访问 ollama.com 下载
  2. 拉取推荐模型
    日常助手: ollama pull mistral-small-3:latest
    高质量回复: ollama pull qwen3:14b
    轻量快速: ollama pull llama3.2:3b
  3. 安装 OpenClaw
    curl -L https://openclaw.com/install.sh | sh
    或参考官方文档配置
  4. 配置 OpenClaw 使用 Ollama
    编辑配置: openclaw config set models.providers.ollama.baseUrl "http://127.0.0.1:11434"
  5. 设置向量数据库（可选，推荐用于文档问答）
    安装 PostgreSQL + pgvector，或用轻量替代品
    配置 OpenClaw 文档索引技能
  6. 配置 Cloudflare Tunnel（外网访问）
    cloudflared tunnel --url http://localhost:3000
    零成本内网穿透，Mac Mini 服务可安全暴露到外网
  7. 配置定时任务和自动化
    设置 cron 任务定期抓取信息源、生成报告
    OpenClaw skill 配置定时触发

💡 关键洞察: 用户从"在 Terminal 里折腾"到"躺在沙发上给 AI 发短信"的转变， 是 OpenClaw 粘性的核心时刻。Mac Mini 24/7 运行的特性让这个体验成为可能—— 你可以随时发消息，AI 随时响应，不需要每次重新启动。

推荐配置

  • Mac Mini M4 16GB: 跑 7B Q4 模型，适合轻量日常助手（~18-22 tok/s）
  • Mac Mini M4 32GB: 跑 14B Q4 模型，质量明显提升，多任务无压力
  • Mac Mini M4 Pro 64GB: 跑 30B Q4 模型，12-18 tok/s，企业知识库首选
